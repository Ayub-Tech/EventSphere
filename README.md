# 🎯 EventSphere – Event Management Platform

En fullstack event-plattform byggd med .NET 8 och Clean Architecture.
Projektet är en skoluppgift och demonstrerar modern backend-utveckling
med REST API, JWT-autentisering och Azure SQL-databas.
